## TONE — 基調と印象

深い藍紫を基調とする。背景では発光するオーラ（バイオレット・ブルー・シアン）が左上から右下へゆったり流れ、星がまたたく。

**印象**：未来・宇宙・前向き。冷たくならないよう発光的な光で体温を保つ。

### 不変の規律

- フラットが基盤。強い影・派手なグラデーションは使わない（わずかな発光のみ）。
- オーラは「沈む」のではなく「灯る・昇る」。
- アクセント（燐光＝シアン `#60CEE0`）は面積を絞り、**要所に一点だけ**（CTA・選択・作品の縁・呼吸）。
- 赤点（通知の未読バッジ）は機能記号の例外。世界観のシアン一点とは層が別。

## BLUR — ぼかしの区別（最重要）

ぼかしの有無を取り違えると世界観が崩れる。

| 要素 | ぼかし | 正体と実装 |
|------|--------|-----------|
| オーラ（サムネ周囲） | **あり** | 作品色のにじむ発光。`box-shadow` の blur で表現（46px / 92px）。サムネ個別に付く。 |
| クリアガラス（サムネ前面） | **なし** | 斜めの光沢のみ。`backdrop-blur` は使わない。画像をぼかさず質感だけ重ねる。 |
| フロスト（トランスポート） | **あり** | すりガラス。`backdrop-filter: blur(14px)`。トランスポート専用。作品サムネには使わない。 |
| ヒーロービジュアル | **なし**（発光） | ぼかしではなく radial の発光が明滅。画像はぼけない。 |
| 背景オーラ | **あり** | 画面最背面で流れる大きなぼかし光（バイオレット/ブルー/シアン）。 |

**一言ルール**：作品の画像そのものは決してぼかさない。ぼかしてよいのは、サムネの「外周のオーラ」「背景」「トランスポートのフロスト」だけ。クリアガラスとヒーローは画像を鮮明に保つ。

## ARTWORK FRAME — カード構造（実装値）

全画面共通。**2:3の角丸矩形「額装カード」**。外側から五層。基準サイズ **152×228**（小サムネ・グリッドは同比率で縮小）。

### 五層構造

| 層 | 実装値 |
|----|--------|
| ① オーラ | `box-shadow: 0 0 46px 6px var(--glow), 0 0 92px 18px var(--glow2), 0 16px 34px rgba(0,0,0,.42)`。glow/glow2 ＝作品の主要色（実装時に自動抽出または運営登録）。ぼかしあり。 |
| ② 台紙（ベース） | `border-radius: 18px` ／ 背景 `linear-gradient(140deg, rgba(255,255,255,.028), rgba(255,255,255,.006))`（ほぼ透明。`#222445` でベタ塗りしない。濃くするとチープ化）／ 縁の光 `inset 0 1px 0 rgba(255,255,255,.20), inset 0 0 0 1px rgba(255,255,255,.05)` |
| ③ 画像 | 台紙内側 `inset: 7px` ／ `border-radius: 11px` ／ `object-fit: cover` ／ シアン縁 `inset 0 0 0 1px rgba(96,206,224,.5)`。画像は台紙よりワンサイズ小さく＝台紙が外周に覗いて額装に見える。 |
| ④ クリアガラス | 下記「クリアガラス」参照（色相不変・ぼかしなし）。 |
| ⑤ ヒーロー | 下記「ヒーロービジュアル」参照（発光の明滅）。 |

### サイズ別

| サイズ | 用途 | 台紙radius | 画像inset/radius | オーラ |
|--------|------|-----------|-----------------|--------|
| 大 152×228 | ディスカバー/プレイヤー/購入 | 18 | inset7 / radius11 | 46px/92px |

## CLEAR GLASS — クリアガラス（全画像共通フィルター）

全作品の前面に共通の光沢層を重ね、シリーズで色が違っても質感を統一。色相は変えない（白・透明のみ）。**ぼかしなし**。

```
斜め光沢: linear-gradient(118deg,
  rgba(255,255,255,.6) 0%,
  .2 14%,
  transparent 32%,
  transparent 62%,
  .24 85%,
  .5 100%
)

上辺: linear-gradient(180deg, rgba(255,255,255,.22), transparent 18%)

縁: box-shadow: inset 0 1px 2px rgba(255,255,255,.4),
               inset 0 0 0 1px rgba(255,255,255,.1)

合成: mix-blend-mode: screen
```

- 画像に焼き込まず、表示時に前面レイヤーとして重ねる＝**画像非破壊**。
- 運営が管理画面で種類（クリアガラス／グレイン／マット／ヴィネット）と強度を選択→全画像へ一括・即時反映。
- オリジナル画像は無傷。A/Bテスト可。**確定＝クリアガラス・効果強め**。

## HERO VISUAL — ヒーロービジュアル

画像全体が蛍のようにゆっくり明滅する発光。動画ではなくプログラム制御＝軽量・全曲一貫。中心集中ではなく全体が淡く灯る。白飛びさせない。

```
発光層:
  radial-gradient(circle at 50% 50%,
    rgba(225,242,250,.9) 0%,
    rgba(195,228,245,.72) 58%,
    rgba(175,218,240,.55) 100%
  )
  mix-blend-mode: screen

周期: 6s ease-in-out
ピーク opacity: .4
@keyframes: 0%,42%{0}  64%{.4}  86%,100%{0}
```

- 運営調整：ON/OFF・範囲（全体／上下左右）・広がり（広め／狭め）・間隔・強さ。基本値＝間隔長め・強さ控えめ。
- **必須**：`prefers-reduced-motion` で停止。
- グリッド（コレクション）では負荷と煩雑さ回避のため省略。

## PURCHASE — 購入時アニメーション（実装値）

決済の祝福＝シアンの呼吸 → 星の一斉点火 → 拡大 → 再生。**暖色不使用**・藍紫＋シアン＋白で完結。星は背景固定。

### ① シアンの呼吸（縁の燐光）

カード外周のシアングローが一度だけ深く脈打つ（立ち上がり〜ピーク約0.6秒）。

```
box-shadow:
  0 0 66px 16px rgba(120,232,255,.78),
  0 0 124px 34px rgba(96,206,224,.5),
  inset 0 0 0 1px rgba(120,232,255,.5)

transition: box-shadow .55s ease-in-out
約700msで戻す
```

### ② 星の一斉点火（縁起点）

- 粒数：約340。半径 `0.35 + random*0.65`（小粒）。
- 色：75%が白 `[255,255,255]`、25%が青み `[150,210,255]`。
- 縁起点で伝播：作品の縁からの距離順にソートし、点火が外へ広がる（`reach = min(1, t/0.25)*1.2`）。
- 包絡：`t<0.12` 立ち上がり → `t<0.4` 満天 → 以降ゆっくり減衰。点火〜収束 **約1.8秒**（STAR_DUR=1800ms）。
- ぼけ半減：描画半径 `rad = r*(1+bright*1.1)`、radialGradient 外半径 `rad*1.4`、中間ストップ `0.5` でコアをくっきり。合成 `lighter`（加算）。
- 呼吸のピークに同期して点火。収束後は通常の疎らな明滅へ。

### ③ 拡大トランジション（起点可変・FLIP）

タップしたカードの矩形（`getBoundingClientRect`）→ 全画面中央へ。

```
transform: translate() + scale()
duration: 0.62s
easing: cubic-bezier(.2,.7,.25,1)
transform-origin: top left
```

拡大後、トランスポートをフェードイン（約1s）し自動再生。金額タップ購入は不可＝**明示の購入ボタンからのみ起動**。

## MOTION — アニメーション周期・イージング一覧

| 対象 | 周期 / 時間 | イージング・備考 |
|------|------------|----------------|
| 背景オーラ | ゆったり（やや速め）・複数速度 | 左上→右下へ循環、色が揺らぐ。星はまたたく（控えめ）。 |
| ヒーロービジュアル | 6s | `ease-in-out`・ピーク opacity.4・全体・`reduced-motion` 停止。 |
| 購入ボタンのグロー | 約3s | 呼吸的グロー・星と同期・`scale(1.05)` 以内・点滅は不採用・`reduced-motion` 対応。 |
| シアンの呼吸（購入） | 立ち上がり約0.6s / 戻し約0.7s | `box-shadow .55s ease-in-out`。 |
| 星の点火（購入） | 約1.8s | 縁起点→満天→収束。加算合成。 |
| 拡大トランジション | 0.62s | `cubic-bezier(.2,.7,.25,1)`・起点可変FLIP。 |
| 縦スワイプ確定/着地 | — | しきい値20%＋速度。着地でわずかなオーバーシュート＋短い一回のバイブ、1秒後にプレビュー音。 |
| 横スワイプ（ストーリー） | 1/4で自動・加速 | オーバーシュートなし・バイブなし（同じ曲の中の移動）。 |
| フッターのフェード | 退場 約160ms / 復帰 約0.5s | 縦スワイプ中は退場、着地・スナップバックで復帰。 |
| 表示の立ち上がり | 穏やか | 急がない。全体に「灯る・昇る」。 |

## TRANSPORT — プレイヤーコントロール（フロスト）

```
背景: rgba(150,160,230,.08)
backdrop-filter: blur(14px) saturate(1.3)
縁: inset 0 0 0 1px rgba(96,206,224,.16)
border-radius: 16px
```

- シーク：3px・`#60CEE0` fill ＋ 白ノブ（8px円）
- コントロール：左54pxスペース ／ 中央 再生・停止（40px円・縁 rgba255.4） ／ 右 ループ（on時シアン）
- **最小セット**：再生/一時停止・シーク・時間・ループのみ。EQ・曲送りなし。
- 縦画面固定。

## 実装方針（React Native）

- 最終実装は **React Native**。本仕様のCSS値は方向を示す参考で、そのままは移植しない＝実装時に作り直す。
- グラデーション・ぼかし・発光・アニメーションは **Skia**（`@shopify/react-native-skia`）が最も再現性が高い。アニメ制御は **reanimated**。
- 色値・ブラー量・速度・発光の強さの精度は、実装段階で実機を見ながら詰める。
- カード構造・ヒーロー発光・星点火は Skia 再現に迷いが出やすい＝**参照実装を別途用意推奨**（`components/` 参照）。
