## TONE — 基調と印象

深い藍紫を基調とする。背景では発光するオーラ（バイオレット・ブルー・シアン）が左上から右下へゆったり流れ、星がまたたく。

**印象**：未来・宇宙・前向き。冷たくならないよう発光的な光で体温を保つ。

### 不変の規律

- フラットが基盤。強い影・派手なグラデーションは使わない（わずかな発光のみ）。
- オーラは「沈む」のではなく「**灯る・昇る**」。
- アクセント（燐光＝シアン `#60CEE0`）は面積を絞り、**要所に一点だけ**（CTA・選択・作品の縁・呼吸）。
- 赤点（通知の未読バッジ）は機能記号の例外。世界観のシアン一点とは層が別。

## BLUR — ぼかしの区別（最重要）

ぼかしの有無を取り違えると世界観が崩れる。

| 要素 | ぼかし | 正体と実装 |
|------|--------|-----------|
| オーラ（サムネ周囲） | **あり** | 作品色のにじむ発光。`box-shadow` の blur で表現（46px / 92px）。サムネ個別に付く。 |
| クリアガラス（サムネ前面） | **なし** | 斜めの光沢のみ。`backdrop-blur` は使わない。画像をぼかさず質感だけ重ねる。 |
| フロスト（トランスポート） | **あり** | すりガラス。`backdrop-filter: blur(14px)`。トランスポート専用。作品サムネには使わない。 |
| ヒーロービジュアル | **なし**（発光） | ぼかしではなく radial の発光が明滅。画像はぼけない。 |
| 背景オーラ | **あり** | 画面最背面で流れる大きなぼかし光（バイオレット/ブルー/シアン）。 |

**一言ルール**：作品の画像そのものは決してぼかさない。ぼかしてよいのは、サムネの「外周のオーラ」「背景」「トランスポートのフロスト」だけ。クリアガラスとヒーローは画像を鮮明に保つ。

## CLEAR GLASS — クリアガラス（全画像共通フィルター）

全作品の前面に共通の光沢層を重ね、シリーズで色が違っても質感を統一。色相は変えない（白・透明のみ）。**ぼかしなし**。

```
斜め光沢: linear-gradient(118deg,
  rgba(255,255,255,.6) 0%,
  .2 14%,
  transparent 32%,
  transparent 62%,
  .24 85%,
  .5 100%
)

上辺: linear-gradient(180deg, rgba(255,255,255,.22), transparent 18%)

縁: box-shadow: inset 0 1px 2px rgba(255,255,255,.4),
               inset 0 0 0 1px rgba(255,255,255,.1)

合成: mix-blend-mode: screen
```

- 画像に焼き込まず、表示時に前面レイヤーとして重ねる＝**画像非破壊**。
- 運営が管理画面で種類（クリアガラス／グレイン／マット／ヴィネット）と強度を選択→全画像へ一括・即時反映。新シリーズにも自動適用。
- オリジナル画像は無傷。A/Bテスト可。**確定＝クリアガラス採用・効果は強め**。

## HERO VISUAL — ヒーロービジュアル

画像全体が蛍のようにゆっくり明滅する発光。動画ではなくプログラム制御＝軽量・全曲一貫。  
中心集中ではなく**画像全体がほぼ均一に灯る**。白飛びさせない。

```
発光層:
  radial-gradient(circle at 50% 50%,
    rgba(225,242,250,.9) 0%,
    rgba(195,228,245,.72) 58%,
    rgba(175,218,240,.55) 100%
  )
  mix-blend-mode: screen

周期: 6s ease-in-out
ピーク opacity: .4
@keyframes: 0%,42%{0}  64%{.4}  86%,100%{0}
```

- 運営調整：ON/OFF・範囲（全体／上下左右）・広がり（広め／狭め）・間隔・強さ。基本値＝間隔長め・強さ控えめ。
- **必須**：`prefers-reduced-motion` で停止。
- グリッド（コレクション）では負荷と煩雑さ回避のため省略。

## P2 — ディスカバー（ホーム）画面

1曲ずつ表示。縦スワイプ＝次の曲、横スワイプ＝ストーリー。作品フレームはカード構造。

### 操作仕様

- **縦スワイプ**：上＝次／下＝前（対称）。しきい値＝サムネ縦幅の**20%＋速度判定**。未満はスナップバック。
- 文字（タイトル・短文）：スワイプ量に連動してフェード。着地で**軽い一回のバイブ**、1秒後にプレビュー音（ON時）。曲間は無音。先頭曲の下スワイプはゴム抵抗。
- **購入ボタン**：中央・横128px×縦38px・テキスト12px（視覚控えめ・タップ域44px以上）。色 `#60CEE0` 一点・周囲に呼吸的グロー（約3秒・星と同期）。
- **プレビュー**：スピーカーアイコン。タイトル右端の延長・音量アニメの下（購入バーから分離）。
- **通知ベル**：右上・音量UIの左・白 `#ECEEF7`・未読は小さな赤点。
- Artist名はストーリー画面のみ（ディスカバー・再生画面には出さない）。
- 検索・絞り込み・並べ替え・ページめくりは作らない。

### フッター挙動（実装値）

- 縦スワイプ中はフッターを速くフェードアウト（**約160ms**）、着地・スナップバックで戻す（**約0.5s**）。
- フッター5タブ：ホーム／コレクション／VIP／メディア／設定。VIPシアン強調。

## P2.1 — ストーリー画面

ディスカバーから左スワイプで右から開く（同じ曲の深部へ。曲は替えない）。再生画面の「ストーリー」からも到達。各項目は絶対座標・固定高さのフロストに分離。

- **表示順**：サムネ（中央上部）→ Story → 調律素材 → Artist。
- サムネは**小サイズのカード構造**（台紙・画像・シアン縁・クリアガラス・オーラ）で上部に保持。
- **横スワイプ**：左で開始、画面幅の1/4超で自動・加速（オーバーシュートなし）。未満は戻る。ディスカバーのサムネは位置固定でフェードアウト。戻るは [<<<]＋右スワイプ。
- Story：行数上限・短文は枠内で縦中央寄せ。調律素材：1行最大4つ・最大8個（2段固定）・運営プリセットから選択。
- **Artist（Naoki Oka）はこの画面のみ表示**。タップで作家プロフィール（P5）へ。
- 加速はゆっくり・バイブなし（同じ曲の中の移動）。プレビューON時は横移動中・表示中も音を継続。

## PLAYER — 再生画面

所有曲の本再生。ディスカバーと同じ背景・アニメーション。上部中央にサムネ（カード構造）、下にフロストのトランスポート。縦画面固定。

- トランスポート（**最小セット**）：再生／一時停止・シーク（位置ノブ付き）・現在時間／総時間・ループ（ON/OFF）。
- EQ・曲送り（前後）は置かない。曲間は無音。バックグラウンド再生対応。
- 到達経路：マイコレクション所有曲タップ／購入完了の遷移／ミニプレイヤーのタップ・上スワイプ／ディスカバーの「再生」ボタン。
- 上部の戻り導線は「**ホームへ戻る**」（旧「再生画面へ戻る」から修正）／右に「ストーリー」。

### トランスポート実装値（フロスト）

```
背景: rgba(150,160,230,.08)
backdrop-filter: blur(14px) saturate(1.3)
縁: inset 0 0 0 1px rgba(96,206,224,.16)
border-radius: 16px

シーク: 3px・#60CEE0 fill ＋ 白ノブ（8px円）
コントロール: 左54pxスペース ／ 中央 再生・停止（40px円）／ 右 ループ（on時シアン）
```

本番は Skia で質感を高める。

## PURCHASE — 購入トランジション（起点可変・案B）

決済の祝福演出は**全導線で共通**。「呼吸 → 星の一斉点火 → 全画面へ拡大 → 自動再生」。  
導線ごとに変えるのは演出ではなく**拡大の起点だけ**。暖色不使用・藍紫＋シアン＋白で完結。

### 共通シーケンス

| ステップ | 内容 |
|----------|------|
| ① 決済成功 | 購入チップ・情景の文字がフェードアウト（作品は保持）。 |
| ② シアンの呼吸 | 作品の縁の燐光シアンが一度だけ深く脈打つ（立ち上がり〜ピーク**約0.6秒**）。カード構造では外周シアングローのパルスで表現。 |
| ③ 星の一斉点火 | 呼吸ピークに同期し、作品の縁起点で星が点火→満天→収束（**約1.6〜1.8秒**）。 |
| ④ 全画面へ拡大 | 同じカードが共有要素トランジションで拡大（FLIP）。 |
| ⑤ 自動再生 | トランスポートがフェードイン（約1s）。 |

### 起点可変（FLIP）実装値

```
起点: getBoundingClientRect() → 位置・サイズからデバイス中央全画面へ
transform: translate() + scale()
duration: 0.62s
easing: cubic-bezier(.2,.7,.25,1)
transform-origin: top left

ディスカバー（中央大カード）: 直進
グリッド（小カード）: 途中位置から斜めに移動しながら拡大
```

### 星（実装値）

- 粒数：約340。半径 `0.35 + rand*0.65`（小粒）。
- 色：75%が白 `[255,255,255]`、25%が青み `[150,210,255]`。
- 縁起点で伝播：作品の縁からの距離順にソート、点火が外へ広がる（`reach = min(1, t/0.25)*1.2`）。
- ぼけ半減：描画半径 `rad*1.4`・中間ストップでコアをくっきり。合成 `lighter`（加算）。

### 状態遷移

未所有「購入する ¥2,500」 → 所有後は同じ場所が「再生」へ。  
「購入直後」と「後から所有曲に再会」を同一状態に統一。  
**金額タップ購入は禁止（ストア審査NG）。明示の購入ボタンからのみ起動。**

### シアンの呼吸（実装値）

```
box-shadow:
  0 0 66px 16px rgba(120,232,255,.78),
  0 0 124px 34px rgba(96,206,224,.5),
  inset 0 0 0 1px rgba(120,232,255,.5)
transition: box-shadow .55s ease-in-out
約700msで戻す
```

## ARTWORK GENERATION — 生成サムネ（Midjourney V8）

器（静かな藍紫のギャラリー）と中身（多彩な作品）を分離し、「複数揃えるとなお美しい」収集体験を作る。

- 生成＝Midjourney V8。`--sref` でシリーズの作風・色を一貫させる。
- **純AI生成物は著作権保護されないため、デザイナーの人間による仕上げを必須とする**。
- アスペクト比 **2:3**（896×1344 等）。明瞭なオブジェクト（主役）を持つこと。
- 収集構造の型：タペストリー型／連作変奏型／カラバリ型。シリーズ単位でトンマナは自由（前面のクリアガラスが質感を統一）。
- 作家ポートレートは**円形**（人物は円・作品は正方形/カードで区別）。
- 日本語入力から英語プロンプトを組む支援ツール（プロンプト・スタジオ）を別途用意。画像添付は技術的に無意味のため非搭載、`--sref` で代替。

## 実装サマリー（エンジニア向け）

React Native（最終実装）。グラデ・ぼかし・発光・アニメは **Skia**（`@shopify/react-native-skia`）、アニメ制御は **reanimated**。本仕様のCSS値は方向を示す参考で、実機で精密に詰める。

| 要素 | 実装の要点 |
|------|-----------|
| カード構造 | 2:3。台紙(radius18・ほぼ透明白)＋画像(inset7・radius11・シアン縁)＋クリアガラス＋ヒーロー＋オーラ。サイズ違い(大/小/グリッド)は同比率で縮小。 |
| クリアガラス | 前面レイヤー・画像非破壊。運営管理画面で種類/強度を選択→全画像へ一括即時反映。色相不変（白・透明のみ）。 |
| ヒーロービジュアル | radial発光・screen合成・6s周期・ピークopacity.4・全体・運営調整（範囲/間隔/強さ）・`prefers-reduced-motion`停止。 |
| 購入トランジション | 起点＝タップカードのrect→全画面へtranslate+scale（FLIP）。共通：文字消失→呼吸(外周シアングロー)→星点火(縁起点・約340粒)→拡大→自動再生。 |
