## 0. 商品体系の全体像

| 商品 | 区分 | 価格 | 売っているもの |
|------|------|------|----------------|
| **toC 買い切り** | FOR PERSONAL | **¥2,500 / 曲**（無料トラック ¥0） | 作品の**所有**。全作品が単一価格。サブスクなし。個人利用限定。 |
| **toB 年間ライセンス** | FOR BUSINESS | **¥36,000 / 年**（月換算 ¥3,000） | 店舗・施設での**商用利用権**（期間契約）。楽曲はアプリで個別購入（個人と同一価格）。 |
| **カスタム制作** | ONE-OFF | **価格協議中**（一体提示） | 音源＋物理カードによる一点物の音楽工芸作品。内訳は提示しない。 |

### 設計の三原則

1. **所有は買い切り、利用権は期間契約** — 作品の所有に月額課金するのは不合理（→ toC は買い切り）。商用利用の権利は期間と結びつくのが本質（→ toB は年間契約）。二つは矛盾しない。
2. **作品はすべて ¥2,500 の単一価格** — 個人も店舗も同じ価格で同じ作品を所有する。店舗との差は「ライセンスが商用利用を解錠するか」だけ。作品側に価格の階層は作らない。
3. **カスタム制作は内訳を出さない（一体提示）** — 分解した瞬間に各要素が市場相場と比較され評価が崩れる。工芸・美術の評価体系に置く。見積書・請求書に至るまで内訳は記載しない。

## 2. toB 年間ライセンス（確定）

| 項目 | 内容 |
|------|------|
| 価格 | **¥36,000 / 年**（月換算 ¥3,000） |
| 売っているもの | 店舗・施設における**商用利用権**（期間契約）。**楽曲は含まれない** — 楽曲は toC アプリで 1 曲 ¥2,500 を個別購入（個人と同一の価格・窓口。運営は経由させない）。 |
| 契約単位 | **OS アカウント 1 つにつき 1 契約**。店舗数・面積等での管理は行わない。 |
| 契約窓口 | **toC アプリ内から契約**。導線は**設定画面の奥など静かな位置**（個人向け空間を法人商品で濁さない）。※ 具体配置は UI 協議で確定。 |
| 特典 | **年次の選曲調律** — 更新時に店舗の空間に合わせた選曲構成を岡ナオキが提案。**店舗の所有曲のみで構成**し、未購入曲は提案に含めない（購入誘導・営業性を排除）。所有曲が増えるほど厚くなる蓄積比例型。 |
| 呼称ルール | 対外的には「年間ライセンス」「施設契約」。**「サブスク」という語は使用しない**（toC の「サブスクなし」との見かけ上の矛盾を避ける）。 |

**価格 ¥36,000 の考え方**: 下限＝店舗BGMサービス相場（月額 ¥3,000 前後〜）を下回らない。上限＝小規模店舗オーナーが即決できる範囲。帯域 ¥36,000〜48,000 の下端からスタートし、初期 10〜20 契約を価格発見期間とする。**改定時は既存契約者を据え置き、新規のみ変更**。

## 三商品の関係（導線）

```
toC アプリ ──→ 年間ライセンス ──→ カスタム制作
（作品を所有）  （商用利用の解錠）   （関係の先の一点物）
```

**導線の思想**: 営業行為でワンオフへ誘導せず、「関係が続いた先にカスタム制作が存在する」状態を作る。煽らない・売り込まない原則を toB 導線でも同一に適用する。

## 4. 無断商用利用への対応方針（確定）

- **監視・追跡はしない**。利用時間の計測等による洗い出しは生産性が低くブランドにも反する。少数の確信犯は放置が最適。
- **構造で正規側へ誘導**: ①規約に商用利用の範囲を明記。②事業者は経費処理のため事業者名義の契約・適格請求書を必要とするため、年間ライセンスに請求書・契約書を標準装備すれば経理上の合理性が正規契約を選ばせる。
- **発覚時は請求・警告ではなく、年間ライセンスを静かに案内**する。法的根拠（規約）は持った上で、対応トーンは煽らない原則の内側に置く。
